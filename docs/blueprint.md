# **App Name**: ChopChop

## Core Features:

- Restaurant Registration and Subscription: Allows restaurants to register on the platform and subscribe to a monthly plan via Flutterwave.
- Online Ordering with Bank Transfer: Enables customers to place orders online and pay via manual bank transfer, uploading payment proof for restaurant confirmation.
- Dine-In Ordering via QR Codes: Generates QR codes for each table, allowing customers to scan and place orders directly from their table.
- Pre-Order System: Allows customers to pre-order food for a specific arrival time, with options to pay via transfer or at the restaurant.
- Restaurant Dashboard: Provides restaurant owners with a dashboard to manage orders, menus, tables, and subscriptions.
- Realtime Order Confirmation Tool: An AI powered tool which notifies restaurants in realtime when new orders are placed or payment proofs are uploaded and will learn which kinds of inputs tend to suggest a confirmation is likely, which require special scrutiny, or which suggest potential fraud.
- Automated Subscription Management: Manages SaaS subscriptions, including payment confirmation via webhook, updating subscription status, and disabling expired restaurants via cron job.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7800) to reflect warmth, appetite, and hospitality.
- Background color: Light orange (#FFEAD3), a softer shade of the primary color.
- Accent color: Analogous reddish-orange (#FF5733) to give contrast.
- Body and headline font: 'PT Sans', sans-serif, combining a modern look with some warmth
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, clear icons representing food categories, order status, and dashboard functions.
- Mobile-first, responsive design with a clean, modern layout optimized for low bandwidth environments.
- Subtle loading animations and toast notifications to provide feedback without overwhelming the user.
